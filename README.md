# Ollama Chatbot

A simple chatbot interface built with Gradio and LangChain for interacting with Ollama models.

## Requirements

- Python 3.8+
- Ollama running locally

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python run.py
```

The application will start on `http://localhost:7860`

## Features

- Chat with local Ollama models
- Custom system prompts
- Model selection
- Conversation history
